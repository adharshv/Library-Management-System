Library Management System:


1. cd into the project cd .../mylibrary/library from the command prompt
2. create a virtual environment
3. install django using pip installer
4. install psycopg2 using pip installer
5. Open Visual Studio here and runserver
6. Go to the url: localhost:8000 which is the homepage of the library management system

OS : built on Windows 10

IDE used : Microsoft Visual Studio Code
Version: 1.31.1

Languages used : postgresql, python 3.6, html, css
Additional dependencies : psycopg2, django
